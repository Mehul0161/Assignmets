
# Assignment 6: Exception Handling

This assignment covers safe programming practices using Python's exception handling system.

## Tasks Included:
1. **Safe Division Utility** - Handling `ValueError` and `ZeroDivisionError`.
2. **Bill Calculator** - Iterating through list and handling `TypeError` and `ValueError`.
3. **Age Validator** - Using `raise` for custom error messages.
4. **File Reader** - Handling `FileNotFoundError` and `PermissionError`.
5. **Safe Shopping Cart** - Continuous input loop with error management.

## How to Run:
Run the script using:
```bash
python exception_handling.py
```
Follow the interactive prompts for Tasks 1, 3, 4, and 5.
