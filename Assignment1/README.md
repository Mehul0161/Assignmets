# Assignment 1: Python - Data Structures

## Overview
This project is an inventory utility for a small e-commerce store, demonstrating the correct use of Python's core data structures: **Lists, Tuples, Sets, and Dictionaries**.

## Project Files
- **`data_structures_assignment.py`**: The main script containing the implementation for all four tasks.
- **`README.md`**: This file, providing an overview of the assignment.

## Task Implementation Details

### Task 1: Product Collections
- Manages product names in a **List**.
- Stores individual product details in a **Tuple**.
- Demonstrates list indexing, appending, and list-to-tuple conversion logic.

### Task 2: Categories
- Creates a **Set** of unique categories.
- Demonstrates adding new categories and checking for existence (membership testing).
- Shows that duplicates are automatically ignored by sets.

### Task 3: Product Pricing
- Uses a **Dictionary** to map product names (keys) to their prices (values).
- Includes logic for adding new products, updating existing prices, and safely removing products.
- Calculates and prints the average price and identifying maximum/minimum priced items.

### Task 4: Combined Operations
- Integrates all data structures by creating a **Catalog** (list of tuples).
- Maps categories to lists of product names.
- Identifies and prints the category containing the highest number of products.

## How to Run
Ensure you have Python installed and run the following command in your terminal:
```bash
python data_structures_assignment.py
```
