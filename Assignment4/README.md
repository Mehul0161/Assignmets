
# Assignment 4: File Handling (Read, Write, Append, Modes)

This assignment covers opening files in different modes, reading with various methods, writing/appending data, and performing basic analytical operations on file contents using Python.

## Tasks Included:
- **Task 1: Write Sales Records to a File** - Creating `sales_data.txt`.
- **Task 2: Read File in Different Ways** - Demonstrating `.read()`, `.readline()`, and `.readlines()`.
- **Task 3: Append New Sales** - Updating `sales_data.txt` with new records.
- **Task 4: Generate Summary Report** - Calculating Total, Highest, Lowest, and Average sales.
- **Task 5: Create Product Info** - Taking user input to create `products.txt`.
- **Task 6: Read File Safely** - Implementing error checking with `os.path.exists()`.
- **Task 7: Mini Project - Export Discounted Prices** - Calculating discounts and generating a report in `discount_report.txt`.

## How to Run:
1. Open a terminal/command prompt.
2. Navigate to this folder.
3. Run the script:
   ```bash
   python file_handling_assignment.py
   ```
4. Follow the on-screen prompts for Tasks 5, 6, and 7.

## Restrictions Followed:
- No `pandas`, `csv` module, or external libraries used.
- Strictly used basic Python file handling features.
- Implemented `with open()` blocks for safe file operations.
