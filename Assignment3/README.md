# Assignment 3: Python - Functions

This assignment covers user-defined functions, recursion, lambda functions, and higher-order functions like `map()` and `filter()`.

## Features
- **Task 1**: Basic function for discount calculation with default arguments.
- **Task 2**: Recursive function for calculating factorials.
- **Task 3**: Lambda functions for GST and final price calculations.
- **Task 4**: Use of `map()` to apply GST across a list of prices.
- **Task 5**: Use of `filter()` to categorize products by price.
- **Task 6**: Combined utility for bulk price processing.
- **Task 7**: Interactive menu-driven utility using modular functions.

## How to Run
Execute the main script in your terminal:
```bash
python functions_assignment.py
```
